My emacs configurations
=======================

This is my emacs configuration. I'm using Carbon Emacs on a MacBook, however this configuration should work on Linux as well.

Installation
------------

    $ mkdir projects
    $ git clone git://github.com/adolfosousa/emacs-config.git
    $ rake
